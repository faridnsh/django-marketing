# Django Audience Jobs-to-be-Done Research

This directory contains evidence-based audience research only. It intentionally excludes marketing inventory, competitor analysis, positioning, messaging, campaign planning, channel strategy, asset recommendations, and website redesign recommendations.

Start with the [deliverables index](INDEX.md).

## Deliverables

- `audiences/`: one research report per audience lens
- `synthesis/`: cross-audience synthesis, taxonomy, segmentation changes, evidence gaps, primary-research plan, and carry-forward hypotheses
- `review/`: independent critical review

Run `./research/validate.sh` for structural checks. The script verifies files, required headings, evidence-table shape, and a minimum citation count; it does not establish that a claim is true or that a source is representative. Those judgments remain in the reports and independent review.
